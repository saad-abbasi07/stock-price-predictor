# Stock Price Movement Predictor

This project predicts next-day stock prices using LSTM.

## Features
- Uses historical stock data from Yahoo Finance
- Trains an LSTM model on multiple companies
- Predicts next-day closing price
- Visualizes actual vs predicted prices

## Files
- `Stock_Price_LSTM.ipynb`: Notebook with data preprocessing, model training, and predictions
- `stock_lstm.keras`: Trained LSTM model
- `requirements.txt`: Python dependencies
